# Digital Escape Room

A console-based escape room game built in C++ using Object-Oriented Programming concepts.

## Gameplay
- Enter your name and explore 5 unique rooms
- Solve puzzles to unlock doors and progress
- Wrong answers reduce your health by 15%
- Collect reward items along the way
- Escape all rooms before your health hits 0!

## Puzzle Types
- **MCQ Puzzle** - Multiple choice questions (10 pts)
- **Logic Puzzle** - Numeric answer puzzles (15 pts)
- **Riddle** - Word riddles with hints (20 pts)

## OOP Concepts Used
- Classes and Objects
- Inheritance
- Polymorphism
- Abstract classes (pure virtual functions)
- Dynamic memory management

## Rooms
1. Dusty Library
2. Clockwork Chamber
3. Whispering Vault
4. Lab of Echoes
5. Final Vault

## How to Run
1. Open in Dev-C++ or any C++ compiler
2. Compile and run main.cpp

## Language
C++ (OOP)
